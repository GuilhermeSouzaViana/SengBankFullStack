Como faço para alterar meus dados ?

Basta ir até a área de perfil, solicitar alterar dados e escolher qual dado deve ser alterado

Qual meu limite de empréstimo ?

O limite máximo de empréstimo não pode ultrapassar o dobro do sálatio base do cliente, ou seja, se o sálario base do cliente for R$1500,00 reais ele poderá solicitar até R$3000,00 reais de empréstimo

Como faço para depositar fundos na conta ?

Basta ir até a área de deposito, selecionar a conta, agencia e o valor desejado para deposito.

Quais são os meus custos no SengBank ?

Os custos no SengBank são apenas para transferências via TED e DOC, sendo via TED de R$11,00 reais e DOC R$13,00 reais.
